# JOB-FINDER

## 目的
2時間の通勤の壁を越え、快適なフルリモート環境で高待遇かつ自身の技術力（これまでの30年にわたるシステム開発の経験）を最大限発揮できる案件を「自動で見つけ出し、的確に応募する」ための戦略的AIシステム。

## プロジェクト構成

- `crawlers/`: Google検索APIや各種求人サイトを巡回するエンジンの格納場所。
- `brain/`: Gemini（AI）を使用し、収集した案件と自身の経歴（Linux, C, Python, VBA, PHP等）をマッチング分析するロジック。
- `templates/`: AIが自動生成する、案件ごとのパーソナライズされた応募文テンプレート。
- `data/`: 抽出された案件リストや、ステータス管理用のCSV。

## A. ターゲット・クライテリア
- **契約形態**: 業務委託のみ。（※派遣案件は完全に除外）
- **勤務形態**: 完全リモート（フルリモート）必須。
- **職種範囲**:
  - 30年の経験を活かすITコンサル・リードエンジニア（C, Python, PHP, VBA）。
  - ITリテラシーを武器にしたテクニカルなオンライン秘書。
  - 映像制作スキルを活用した動画編集案件。
- **NG条件**:
  - 派遣契約の案件
  - 常駐必須
  - ITリテラシーが著しく低い環境
  - 低単価の単純作業

## B. 主要機能
1. **自動探索 (Sojourner)**
   - Google Custom Search API などを利用し、複数の求人媒体から「自身向け」の新着案件を自動でキャッチする。
2. **適合度判定 (Analyzer)**
   - 自身のポートフォリオをGeminiに読み込ませ、各案件の「採用可能性」と「環境の良さ」をA〜Eランクで判定する。
3. **営業文生成 (Salesman)**
   - 判定結果に基づき、自身の実績（例えば「神戸のクラブチーム向けDX支援『コベカツ』」等）を具体例として盛り込んだ、パーソナライズされた応募メッセージをAIが自動で作成する。
